# 安装编译说明

## 环境预备
1. 安装nvm和iojs。
2. 安装依赖。

## 自动更新CSS
``` bash
gulp sass
```

## 自动化测试
``` bash
gulp test
```

## I18N
``` bash
gulp pot
gulp translations
```
